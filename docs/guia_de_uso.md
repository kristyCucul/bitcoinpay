# BitcoinPay - TPV Lightning para emprendedores

Proyecto que permite a negocios y emprendedores aceptar Bitcoin Lightning usando LNbits de una forma facil y sencilla, de esta forma lograremos el mayor uso de Bitcoin en El Salvador no solo en grandes comercios sino en todo tipo.

# Funcionalidades
- Generación de invoices
- Código QR dinámico
- Pagos con Zeus Lightning Wallet
- Verificación del estado del pago
- División de pagos (Split Payments LNbits)

# Tecnologías
- Python (Flask)
- HTML, CSS, JS
- LNbits API
- Zeus Wallet

## Cómo ejecutar

### Backend
cd backend  
pip install -r requirements.txt  
python app.py
pip install flask-cors

### Frontend
Abrir frontend/index.html
